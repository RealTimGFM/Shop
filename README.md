# Shop
This is a simple iOS shopping app using SwiftUI and Firebase.  
It lets users or shop owners sign in, browse products, add, edit and delete products or stores.

## Features
- Email/password authentication with Firebase
- Real-time data from Firestore
- Create, read, update, delete products or stores

## Tech Stack
- Swift
- Firebase
- Xcode / iOS

## Getting Started

1. Open project in Xcode.
2. Get GoogleService-Info.plist from Firebase and add to the project
3. Configure Firebase is in ShopApp.swift.
4. Run the app.

This project is only for education purposes.
